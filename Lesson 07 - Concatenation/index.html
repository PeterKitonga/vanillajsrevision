Concatenation

Next >>

In JavaScript, concatenation is done using the `+` sign to join different data type variables to a string. This may affect any additions that one may try to apply in the string. To avoid this wrap the numbers in question in brackets. However, it is better to first do the addition before concatenating.

Code:

  1. var name = 'Peter',
  2.     age = 26,
  3.     married = false;
  4.  
  5. console.log('My name is ' + name + ' and in 2 years I will be ' + (age + 2) + '. Am I married? ' + married);
