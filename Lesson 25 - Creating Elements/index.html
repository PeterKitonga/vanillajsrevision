Creating Elements

Next >>

In JavaScript, we can create elements and append them to already existing elements in the DOM. This provides a dynamic way of adding elements to the DOM. We achieve this by using the createElement() function. Once the element is created we append it to an elements using the appendChild() function.

Code:

  1. var app = {};
  2.  
  3. app.container = document.querySelector('.result-container');
  4. app.button = document.querySelector('.result-button');
  5.  
  6. // Listen for the on click event, create and append elements to the container
  7. app.button.addEventListener('click', function() {
  8.     var spans = document.querySelectorAll('.result-element');
  9.  
  10.     var element_number = spans.length + 1;
  11.  
  12.     // Creates an element in the DOM
  13.     app['element_'+element_number] = document.createElement('span');
  14.  
  15.     // Sets the attributes of the element created
  16.     app['element_'+element_number].setAttribute('class', 'result-element');
  17.  
  18.     app['element_'+element_number].innerText = 'Created element ' + element_number
  19.  
  20.     app.container.appendChild(app['element_'+element_number]);
  21. });
