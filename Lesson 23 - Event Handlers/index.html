Event Handlers

Next >>

In JavaScript, event handlers are blocks of code that run when an event is triggered in the DOM e.g clicks, hovers, focus. These handlers listen for a specific event on a specified element and run a block of code.

Code:

  1. var app = {};
  2.  
  3. app.elements = document.querySelectorAll('a.custom-nav-link');
  4.  
  5. /*
  6. * Here we loop because the querySelectorAll method returns
  7. * all elements with the defined selector as an array.
  8. * Then within the loop we add the event handler for each element.
  9. */
  10. app.elements.forEach(function (element) {
  11.     element.onclick = function() {
  12.         console.log('Example #1 click event handler result: ' + this.text);
  13.     }
  14. });
  15.  
  16. /*
  17. * Here I used a different way to add event handlers which is
  18. * now defined and executed via JavaScipt unlike the above which executes through
  19. * the DOM.
  20. */ 
  21. app.elements.forEach(function (element) {
  22.     element.addEventListener('mouseenter', function (event) {
  23.         console.log('Example #2 mouseenter event handler result: ' + event.target.getAttribute('href'));
  24.     });
  25. });
  26.  
  27. /*
  28. * The above way of adding event handlers is not supported in Internet Explorer with
  29. * versions 8 and below. A solution to this would be to attach the event like so:
  30. * app.elements.forEach(function (element) {
  31. *     element.attachEvent('onclick', function () {
  32. *         // Logic Here
  33. *     });
  34. * });
  35. */
