Minification

It is good practice to minify JavaScript code as this helps reduce the file size of your code and helps improve loading time of your web page. In this lesson I used refresh-sf.com to minify my code. Minification works by compressing your javascript file into a .min.js file. The code itself is minified into a single line of code. For maintainability, you can also provide the unminified file for other developers to modify and customize to their liking.
