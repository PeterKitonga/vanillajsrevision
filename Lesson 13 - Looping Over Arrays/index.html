Looping Over Arrays

Next >>

In JavaScript, a `loop` provides an easy way to iterate through data repeatedly and access values in an array or object. Looping over arrays provide a way to access the array index values to use or manipulate them. There are various loop structures and iteration mechanisms in javascript. They are well documented here >. The most commonly used way to loop an array is the `for` loop. Others such as the `for...of` can be used as well. A `for...in` loop can be used but is not recommended for arrays but rather for objects.

Code:

  1. var names = ['Peter', 'Wayne', 'Fred', 'Jax', 'Stacy'];
  2.  
  3. /* *
  4. * Since arrays are 0 indexed, the second parameter(condition) of the for loop is a constraint to prevent the loop from iterating outside the bounds of the array.
  5. * 
  6. * The `names.length` will dynamically pick the number of items in the array and update the condition that constraints the iteration.
  7. * */ 
  8. for(var index = 0; index < names.length; index++) {
  9.     console.log('for loop index ' + index + ': ' + names[index]);
  10. }
  11.  
  12. // this loop structure is different since you can access the array values directly without specifying the index
  13. for (var name of names) {
  14.     console.log('for...of loop value: ' + name);
  15. }
  16.  
  17. var car = {name: "Range Rover Velar", year_of_manufacture: 2019, price: 'USD 200,000', engine_no: 'SHHDSJD/SHDHJS/Q2829'};
  18.  
  19. /* *
  20.  * This loop is used to iterate enumerable properties of an object. It is mostly used to iterate over objects with a key/value pair structure.
  21.  * 
  22.  * It may be most practically used for debugging purposes, being an easy way to check the properties of an object.
  23.  * */ 
  24. for(var key in car) {
  25.     console.log('for...in loop key "' + key + '": ' + car[key]);
  26. }
  27.
