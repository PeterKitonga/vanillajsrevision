Intervals

Next >>

In JavaScript, intervals are created using the setInterval function which repeatedly calls a function or executes a snippet of code within a fixed time delay. This interval is usually cleared by using the clearInterval function whose parameter will be the variable assigned to the interval. More info here >.

Code:

  1. var app = {
  2.     number: 1,
  3.     add: function () {
  4.         return this.number++;
  5.     }
  6. };
  7.  
  8. var interval = setInterval(function () {
  9.     console.log('Interval result: ' + app.number);
  10.     app.add();
  11.  
  12.     if (app.number > 10) {
  13.         // clears the interval
  14.         clearInterval(interval);
  15.         console.log('Interval cleared');
  16.     }
  17. }, 1000);
