In JavaScript, every value is an object except primitive values. Primitive values are values that have no properties or methods. They are usually hard coded and cannot be changed. If x = 3.14, you can change the value of x. But you cannot change the value of 3.14. With that in mind, this means that objects are variables but objects can contain many values. This is where object literals come in and are defined as comma-separated lists of name-value pairs.

Code:

  1. var name = 'Peter',
  2.     names = ['Peter', 'Wayne', 'Fred', 'Jax', 'Stacy'],
  3.     age = 26,
  4.     app = {
  5.         name: 'JavaScript',
  6.         version: 6
  7.     };
  8.  
  9. /*
  10.  * JavaScript mainly works with objects. Each variable with a value of a string, number, date, array 
  11.  * is usually an object with various properties and functions available to it.
  12.  * More info at https://developer.mozilla.org/en-US/docs/Web/JavaScript/Guide/Working_with_Objects
  13.  */
  14. console.log('String Object result: ' + name.length);
  15. console.log('Array Object result: ' + names.indexOf('Fred'));
  16. console.log('Number Object result: ' + age.toExponential(2));
  17.  
  18. /* 
  19. * A JavaScript object literal is a comma-separated list of name-value pairs wrapped in curly braces.
  20. * More info: https://www.dyn-web.com/tutorials/object-literal/
  21. */
  22. console.log('Object literal result: ' + app.name);
