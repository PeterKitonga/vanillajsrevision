Array Prototype

Next >>

In JavaScript, a `prototype` is an object with a constructor property which points back to the function on which the prototype object itself is a property to.The array prototype constructor allows one to add methods and properties to the Array() object. This allows the Array() object to also inherit properties and features from the Array.prototype constructor. For further info on prototypes in JavaScript, check out here > There are various methods and properties that are made available to the Array() object through the Array.prototype such as find(), filter() e.t.c. These are listed and documented in the Mozilla MDN web docs here >

Code:

  1. var names = ['Peter', 'Wayne', 'Fred', 'Jax'];
  2.  
  3. // here we are able to see that by default every Array() object has a length property
  4. console.log(Object.getOwnPropertyNames(names));
  5.  
  6. // this property points back to the Array.prototype of the names Array() object and its available methods
  7. console.log(names.__proto__);
  8.  
  9. // through inheritance we can access these available methods using the dot notation
  10. names.push('Stacy')
  11. console.log(names);
