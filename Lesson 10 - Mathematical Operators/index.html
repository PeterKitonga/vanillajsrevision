Mathematical Operators

Next >>

In JavaScript, arithmetic/mathematical operators are used with numerical values to caculate and return a single numerical result. The arithmetic operators include:

  1. Addition +
  2. Subtraction -
  3. Division /
  4. Multiplication *
  5. Reminder %
  6. Increment ++
  7. Decrement --
  8. Exponentiation **
For more comprehensive info, check out here >

Code:

  1. var number_one = 100,
  2.     number_two = 50,
  3.     result = 0;
  4.  
  5. result = number_one + number_two
  6. console.log('Addition result: ' + result);
  7.  
  8. result = number_one - number_two;
  9. console.log('Subtraction result: ' + result);
  10.  
  11. result = number_one / number_two;
  12. console.log('Division result: ' + result);
  13.  
  14. result = number_one * number_two;
  15. console.log('Multiplication result: ' + result);
  16.  
  17. // this operator is mostly used in loop structures to increment the index of a for loop
  18. number_one++;
  19. console.log('Increment result: ' + number_one);
  20.  
  21. // this operator is mostly used in loop structures to decrement the index of a for loop
  22. number_two--;
  23. console.log('Decrement result: ' + number_two);
  24.  
  25. result = number_one % number_two;
  26. console.log('Reminder result: ' + result);
  27.  
  28. // this operator can also be used to determine wether a number is even by dividing it by 2. If the result is 0, it is even
  29. result = number_one % 2 === 0;
  30. console.log('Even number (' + number_one + ') check result: ' + result);
  31.  
  32. // this will follow the BODMAS order of operations
  33. result = number_two / (number_one % number_two) * number_two + number_one;
  34. console.log('Combined operators result: ' + result);
  35.  
  36. // this operator is used to raise the numerical value by the power of the numerical value to the right of the operator
  37. result = number_two ** 2;
  38. console.log('Exponentiation result: ' + result);
