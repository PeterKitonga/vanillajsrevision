As JavaScript developers, it is good practice for us to lint our code. This improves readability and maintainability of our code. It helps provide a standard in which we write our code. In this lesson, I learnt how to lint my code using jslint.com. One can also install plugins to his/her IDE/Text editor of choice that lint code.
