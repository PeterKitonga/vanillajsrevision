In this lesson, I got to understand that when one declares a variable it becomes global which can be seen by typing `window` in the console where the variable and its value is visible. If another javascript file has a variable with the same name, it can cause issues with functionality in the application. Also it is good practice to declare all variables in a comma separated declaration as illustrated below instead of declaring `var` over and over again.

Code:

  1. var name = 'Peter',
  2.     gender = 'Male',
  3.     age = 26,
  4.     married = false,
  5.     hair_color,
  6.     nameAgain = 'Kitonga';

Result: Open console and type variable name
