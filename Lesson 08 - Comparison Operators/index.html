Comparison Operators

Next >>

In JavaScript, comparison operators are used tp compare values of variables. They can be used to run certain sections of code when the conditions are satisfied. There are a number of comparison operators as follows:

  1. Equality ==
  2. Inequality !=
  3. Strict equality ===
  4. Strict inequality ===
  5. Greater than >
  6. Less than <
  7. Greater than or equal to >=
  8. Less than or equal to <=
For more comprehensive info, check out here >

Code:

  1. var name = 'Peter',
  2.     age = 26,
  3.     fathers_age = 50;
  4.  
  5. // works with variables of different data types. Checks and converts variable data type before comparison
  6. console.log(age == '26');
  7. console.log(age != '26');
  8.  
  9. // strict, only works with vairables of the same data type
  10. console.log(age === '26');
  11. console.log(age !== '26');
  12.  
  13. console.log(age > fathers_age);
  14. console.log(age < fathers_age);
