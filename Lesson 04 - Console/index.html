In this lesson, I learnt about using the console to print out values/results of our code. The Console is mostly used during debugging of the logic.

Code:

  1. var calculator_prompt = prompt('How much would you like to borrow');
  2. var interest_rate = 0.045;
  3. var principal_amount = parseInt(calculator_prompt);
  4. var interest_amount = principal_amount * interest_rate;
  5. var loan_amount = interest_amount + principal_amount
  6.  
  7. console.log("Principal: "+principal_amount);
  8. console.log("Interest: "+interest_amount);
  9. console.log("Loan Amount: "+loan_amount);

Result: Principal: 2000, Interest: 90, Loan Amount: 2090
