Styling Elements

Next >>

In JavaScript, we can style elements in the page, however, it is not good practice to do so. We are expected to only style on specific occassions like after event triggers. Here we use the style property to pass individual styles to an element. We can also use the setAttribute() function to pass multiple styles to an element.

Code:

  1. var app = {};
  2.  
  3. app.button = document.querySelector('.result-button');
  4.  
  5. // Listen for the on click event and style the button
  6. app.button.addEventListener('click', function(event) {
  7.     var colors = ['red', 'green', 'blue', 'cyan', 'yellowgreen'];
  8.  
  9.     /*
  10.     * The style property is an object with the various 
  11.     * css properties that we can use to style an element.
  12.     */ 
  13.     event.target.style.backgroundImage = 'none';
  14.     event.target.style.backgroundColor = colors[Math.floor(Math.random() * colors.length)];
  15. });
