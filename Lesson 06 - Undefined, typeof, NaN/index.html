In this lesson, I to refresh my memory on the use of typeof operator. This operator is used in JavaScript to determine the datatype of a vairable. I also got to learn that `parseInt` will extract only the first interger in a string and ignore the rest. It will also only extract a number if there are no strings before it e.g. parseInt('F3892') will result in a NaN.

  10. number_one = '100F2932';
